搭建博客Hexo+Matery(一)

Hexo
搭建博客

更新日期:   2021-08-06
阅读次数:  

生成密钥将本地与GitHub连接起来

  • 运行下面命令生成密钥
ssh-keygen -t rsa -C "你的邮箱地址"
  • 连点三次回车出现下面界面，密钥生成成功

  • 这是在C盘user文件夹下生成一个.ssh文件夹，打开里面的id_rsa.pub文件复制全部内容，打开GitHub设置添加ssh

  • 随边起一个名字，将复制到内容添加进去，点击添加即可

  • 然后回到Git的命令行界面，测试一下是否与GitHub连接成功。输入下面的命令行：

    ssh -T git@github.com

  • 点击回车，然后会出现一个询问内容，输入yes，回车，会出现一段内容，Hi ** ! You’ve successfully authenticated, but GitHub doesnot provide shell access.。 说明连接成功。此处这个**应该是你Github的用户名。

本地安装Hexo

  • 首先选择一个磁盘作为你博客文件的存放位置，然后新建一个文件夹，比如名为blog的文件夹，创建完后，在此处点击鼠标右键，选择Git Bash Here，然后依次输入如下命令，
# hexo框架的安装 
npm install -g hexo-cli 
# 等上一个命令完成后，在输入下面的命令 
hexo init
继续执行以下指令 npm install（安装依赖包）
hexo generate（构建）
hexo server（启动服务，关闭服务是ctrl+c）
启动后   输入路径localhost:4000到游览器即可看到效果了，类似这样的网

  • 打开根目录下的_config.yml配置文件
title: 你的博客名 
subtitle: 博客的副标题，有些主题支持 
description: 博客描述 
keywords: 博客关键词 
author: 作者，在文章中显示 
language: 博客语言语种 
timezone: 时区

修改url地址，在16行左右，改为https://github用户名.github.io

url: https://gqilin.github.io

滑到文件最底部，有一个deploy，在deploy下面添加一个repo项 ，一个branch项。填入如下代码，并如下图所示：

deploy:
  type: git
  repo: https://github.com/gqilin/gqilin.github.io.git
  branch: main

==注意：==这里有一个英文状态下的空格

上传到GitHub

先运行下边命令安装一个插件

npm install hexo-deployer-git --save
//然后一次运行
hexo g
hexo d
  • 出现这个提示，点击回车

  • 会在浏览器打开这个界面

  • 点击授权，返回小黑框，出现下面情况代表上传成功

  • 在浏览器打开https://github用户名.github.io出现本地运行时一样的界面

至此，一个简单的博客搭建完成。
